# React + Vite + CRXJS + Tailwind v4 Extension Template

### Features
- **React 18** for UI.
- **Tailwind CSS v4** (Shadow DOM compatible).
- **CRXJS** for Manifest V3 and HMR.
- **Shadow DOM Injection** to prevent style leaking.

### Setup
1. `npm install`
2. `npm run dev`
3. Load the `dist` folder into Chrome (`chrome://extensions`).
